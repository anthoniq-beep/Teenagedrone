# 开发计划：中国青少年无人机科学素养培训网站

由于项目骨架已经存在（React + Tailwind + Router），我们将在此基础上进行开发。

## 1. 依赖安装与配置
*   安装 `@supabase/supabase-js` 用于后端连接。
*   配置 `src/lib/supabase.ts` 客户端初始化文件。
*   **注意**：我将提供 SQL 建表语句，请您在 Supabase 后台 SQL 编辑器中运行以创建数据库结构。

## 2. 路由与页面结构搭建
更新 `src/App.tsx` 以支持新的路由结构：
*   **公开区**：
    *   `/` (首页)
    *   `/courses/detail` (课程详情)
    *   `/training/system` (培养体系)
    *   `/competitions/categories` (竞赛类目)
*   **会员区** (需登录)：
    *   `/member/login` (登录)
    *   `/member` (会员中心布局)
        *   `/member/courses` (课程管理)
        *   `/member/profile` (个人信息)
        *   `/member/progress` (培训进度)
        *   `/member/awards` (获奖墙)

## 3. UI 组件开发 (Phase 1: 首页与公开页)
*   **通用布局**：创建 `Navbar` (含登录入口) 和 `Footer`。
*   **首页**：实现三个视觉化入口卡片（课程、体系、竞赛）。
*   **子页面**：实现课程详情、培养体系、竞赛介绍的静态展示页面。

## 4. 认证与会员功能 (Phase 2: 会员系统)
*   **登录页**：实现用户名/密码登录表单。
*   **权限控制**：创建 `ProtectedRoute` 组件，保护会员路由。
*   **会员布局**：侧边栏导航 + 内容区域。
*   **功能页面**：
    *   课程管理：展示已报名课程列表。
    *   个人信息：表单编辑。
    *   培训进度：进度条/图表展示。
    *   获奖墙：证书网格展示。

## 5. 数据集成
*   将前端页面连接到 Supabase 数据表。
*   处理加载状态和空数据状态。
